# CSS Button Collection

A single-page project that showcases **30 different button styles** built using **pure HTML and CSS**.

Each button demonstrates different visual techniques such as hover effects, borders, gradients, animations, and focus styles.  
This project is useful as a **design reference** and a **learning resource for CSS effects**.

## ✨ Features

- 🎨 30 unique button designs
- ⚡ Pure CSS (no JavaScript)
- 📱 Responsive grid layout
- ♿ Accessible focus states
- 🌙 Dark mode support with `prefers-color-scheme`
- 🧩 Easy to copy and reuse in other projects
